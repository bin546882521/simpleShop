# simple store

## 项目介绍

是一个电商 H5 页面前端项目，基于 Vue3.0 + Typescript + axios + less + vant + vue-router + iconfont 实现，使用 mockjs 数据进行开发，主要包括首页、分类页面、我的页面、购物车等
